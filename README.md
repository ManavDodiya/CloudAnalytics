# Cloud Analytics Dashboard

A responsive, high-performance web dashboard for comparing AI infrastructure pricing and performance metrics across top cloud providers: **AWS**, **Azure**, **Google Cloud**, **Alibaba Cloud**, and **Oracle Cloud**.

## Features

- **Comparative Radar Charts**: Normalized overview of cost, latency, uptime SLA, coverage, and support using interactive Chart.js radar charts.
- **Dynamic Token Cost Estimator**: Calculate projected costs on-the-fly based on model tier, input tokens, and output tokens.
- **Glassmorphism Design**: Premium modern dark-theme aesthetics featuring GSAP animations and translucent layered styling.
- **Live Performance Tabulations**: Toggle and contrast pricing vs. SLA metrics beautifully rendered side-by-side.

## Tech Stack

*   **Vite** - Next Generation Frontend Tooling
*   **HTML5 / CSS3 / Vanilla JS** - Core architecture without heavy frontend dependencies.
*   **Chart.js** - Dynamic interactive charts via HTML5 Canvas.
*   **GSAP** - Professional-grade JavaScript animations.
*   **Lucide Icons** - Clean SVG iconography.

## Getting Started

Follow these steps to set up the project locally:

### 1. Requirements

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation

Clone the repository and install the Vite dependencies:

```bash
git clone https://github.com/ManavDodiya/CloudAnalytics.git
cd CloudAnalytics
npm install
```

### 3. Run Development Server

Start the local development server:

```bash
npm run dev
```

Navigate to `http://localhost:5174/` (or the port specified in your terminal) to view the web dashboard.
